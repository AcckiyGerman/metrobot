Скачать и установить Python3:
	https://www.python.org/ftp/python/3.5.1/python-3.5.1.exe

Установить модуль работы с изображениями Pillow, а также pyautogui.
Для этого в командной строке набрать:
	pip install pillow
	pip install pyautogui

Запустить игру Метро. Игра должна быть видна на экране, чтобы бот мог
кликать туда мышкой.

		--------------ВАЖНО!!!!-------------
Маштаб веб страницы в браузере должен быть выставлен на уровень 100%
Иначе программа-бот не сможет распознавать элементы игрового интерфейса.

Запустить STARTBOT.BAT
Если игровой процесс пошел - идите спать :)
Если нет - вероятно вам нужно заменить изображения в папке img/ на 
аналогичные, сделанные скриншотом из вашего браузера.
